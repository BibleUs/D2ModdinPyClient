D2ModdinPyClient
================

Crossplatform equivalence to https://github.com/kidovate/D2ModdinClient



How to build:

- git clone git@github.com:BloodyD/D2ModdinPyClient.git d2modd
- cd d2modd
- chmod +x install_deps.sh install_sip.sh install_pyqt.sh
- virtualenv .
- source bin/activate
- ./install_deps.sh
- ./install_sip.sh
- ./install_pyqt.sh
